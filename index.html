Publications

Water Consumption Monitoring System and Home Automation Based on LOGO! 8.3 and AWS Cloud Connection
IEEE 2023
Jesus Manuel Vargas-Cruz Carlos Eduardo Vidal-Villeda Carlos Alberto Pedro Camelo-Romero

This paper presents a home automation system designed to promote water conservation using Siemens LOGO! 8.3 modules. The solution integrates three subsystems: water usage monitoring, rainwater harvesting with irrigation, and lighting control. Real-time remote access is enabled via AWS Cloud. The system highlights how technology can assist informed users in sustainable water use.
